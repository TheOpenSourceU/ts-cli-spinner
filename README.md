# ts-cli-spinner

Typescript based CLI Working indicator for Node-Based
CLI applications. 

## Example Use

See [examples/app1.js](examples/app1.js) for a simple example of how to use this package.

## Installation

```sh
npm install ts-cli-spinner
```
